# site-global-guide

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库旨在集中存放和展示多个独立的 HTML 页面。这些页面可能涉及不同主题或功能，彼此之间没有强关联，但统一通过此仓库进行版本管理和发布。

仓库本身不针对任何特定域名或网站，仅作为 HTML 文件的归档与托管空间。

## 目录说明

```
site-global-guide/
├── README.md          # 本文件
├── pages/             # 存放所有 HTML 页面的目录
│   ├── page-a.html    # 示例页面 A
│   ├── page-b.html    # 示例页面 B
│   └── ...
└── assets/            # 公共资源目录（如 CSS、JS、图片等）
    ├── style.css
    └── script.js
```

- `pages/`：所有独立的 HTML 页面文件均放置于此目录下，文件名应具有描述性。
- `assets/`：存放页面可能依赖的公共样式、脚本或图片资源。建议各页面尽量自包含，减少外部依赖。

## 页面归档说明

- 每个 HTML 页面应为一个独立、完整的文档，可在浏览器中直接打开运行。
- 建议页面使用相对路径引用 `assets/` 中的资源，以保持结构清晰。
- 页面内容需符合法律法规及 GitHub 社区准则，不得包含恶意代码或不当信息。
- 提交新页面时，请在 commit 信息中简要说明页面用途或变更内容。

## 维护说明

- 欢迎通过 Pull Request 提交新的 HTML 页面或改进现有页面。
- 提交前请确保页面无明显错误，并在本地测试可正常显示。
- 若需删除或替换页面，请在 PR 中说明理由。
- 仓库维护者保留审核和合并的权利。
